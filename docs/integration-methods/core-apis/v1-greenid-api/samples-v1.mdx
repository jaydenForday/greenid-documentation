# Samples V1

Attached to this page is a ZIP archive called "[samples.zip](https://vixverify.atlassian.net/wiki/download/attachments/8880472/samples.zip?version=1\&amp;modificationDate=1465184462514\&amp;cacheVersion=1\&amp;api=v2)".  

This file contains some sample SOAP envelopes for requests and responses to the API. Each sample includes a "request" file and a corresponding "response" file.  For example "registerVeriifcation\_request.xml" shows a sample SOAP envelope for calling registerVerification, and "registerVerification\_response.xml" shows the response given by greenID.
