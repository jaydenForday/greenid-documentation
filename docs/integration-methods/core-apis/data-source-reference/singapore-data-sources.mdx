## sgcreditheader (version 2)

Also known as Singapore Credit Header

| **Field name**                    | **Description**                                               | **Required** | **Restrictions**                    |
| --------------------------------- | ------------------------------------------------------------- | ------------ | ----------------------------------- |
| greenid\_sgcreditheader\_number   | The Singapore NRIC number.                                    | Yes          | 9 alpha-numeric characters          |
| greenid\_sgcreditheader\_surname  | The person's surname.                                         | Yes          | Length between 1 and 255 characters |
| greenid\_sgcreditheader\_dob      | The person's date of birth.                                   | Yes          | DD/MM/YYYY                          |
| greenid\_sgcreditheader\_postcode | The postcode of the residence the person currently lives at.  | Yes          | 6 numeric digits                    |
