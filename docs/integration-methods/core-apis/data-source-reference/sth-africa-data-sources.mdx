# South African Data Sources

| **Title**      | **Webservice Name** | **Type**       | **Coverage** |
| -------------- | ------------------- | -------------- | ------------ |
| National ID    | nationalidentity    | National ID    | South Africa |
| zacreditheader | compuscan           | Credit History | South Africa |

## nationalidentity (version 1)

Also known as National ID

| **Field name**                        | **Description**                     | **Required** | **Restrictions**               |
| ------------------------------------- | ----------------------------------- | ------------ | ------------------------------ |
| greenid\_nationalidentity\_number     | The person' South African ID number | Yes          | Must be a valid ZA national ID |
| greenid\_nationalidentity\_givenname  | The person's given name             | Yes          | Length between 1 and 255       |
| greenid\_nationalidentity\_middlename | The person's middle name            | No           | Length between 1 and 255       |
| greenid\_nationalidentity\_surname    | The person's surname                | Yes          | Length between 1 and 255       |
| greenid\_nationalidentity\_dob        | The person's date of birth          | Yes          | DD/MM/YYYY                     |
| greenid\_nationalidentity\_country    | ZA                                  | Yes          | ZA for South Africa            |
