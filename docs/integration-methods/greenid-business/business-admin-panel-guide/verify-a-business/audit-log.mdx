# Audit Log

## Procedure 29. View audit logs

On the [Verify a Business](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/492307184/Verify+a+business) screen:

1\. Select **Audit Log**. (See Item **2** in [Figure 4](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/492307184/Verify+a+business#Verifyabusiness-Figure4))

![](/img/audit-1.png)

The Audit Log screen, shown in [Figure 13](https://vixverify.atlassian.net/wiki/pages/resumedraft.action?draftId=494043402#Linkedpersonalverificationrecords-Figure13) below is displayed. Each event is time and date stamped, whether it was a system action or administrator action. If it was an administrator action then the administrator’s email is also shown.

*Figure 13. Audit Log Screen — Audit Logs for the business verification*

![](/img/audit-2.png)

3\. Select **Back to verification** to return to the main [Verify a Business](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/492307184/Verify+a+business) screen.
