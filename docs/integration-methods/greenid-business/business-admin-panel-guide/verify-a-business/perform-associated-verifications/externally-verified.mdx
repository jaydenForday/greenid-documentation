# Externally Verified Associated Parties

On the [Verify a Business](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/492307184/Verify+a+business) screen:

1\. Navigate to the person you want to verify and select the dropdown options button.

![](/img/external-1.png)

2\. Select **Mark as verified externally**.

![](/img/external-2.png)

3\. When the dialog appears, enter a note explaining the verification.

![](/img/external-3.png)

4\. Select **Mark “verified externally”.**
