# Verification Notes

## Procedure 26. Add a verification note

On the [Verify a Business](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/492307184/Verify+a+business) screen:

1\. Select **Add a note**. (See Item 3 in [Figure 4](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/492307184/Verify+a+business#Verifyabusiness-Figure4))

![](/img/notes-1.png)

2\. Enter your comments on the Verification Notes dialog.

![](/img/notes-2.png)

3\. Select Add this note.

## Procedure 27. View verification notes

On the [Verify a Business](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/492307184/Verify+a+business) screen:

1\. Select **Add a note**. (See Item **3** in [Figure 4](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/492307184/Verify+a+business#Verifyabusiness-Figure4))

![](/img/notes-3.png)

The Verification Notes dialog, shown in [Figure 11](https://vixverify.atlassian.net/wiki/pages/resumedraft.action?draftId=494043402#Linkedpersonalverificationrecords-Figure11) below, is displayed. Each note is time and date stamped and indicates who entered it. (Notes are not editable.)

*Figure 11. View/Add Notes — Verification Notes dialog*

![](/img/notes-4.png)

2\. Select **Close**.
