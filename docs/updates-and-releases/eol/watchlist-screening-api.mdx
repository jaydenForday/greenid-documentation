# Watchlist Screening API Response Comparisons

Itâ€™s easy to upgrade from the standard PEP and sanctions service to GBG Watchlist Screening with Dow Jones data, GBG will simply make a configuration change. This will not change the API structure but it will change some of the API responses.

Here is a table comparing the API responses.

| **Standard Watchlist - Results**                                                                                                                                                                                                               | **Watchlist Dow Jones - no matches**                                                                                                                                                                                          | **Watchlist Dow Jones - match on PEP**                                                                                                                                                                                         | **Watchlist Dow Jones - match on Sanctions**                                                                                                                                                                                                           |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| <p>In the standard watchlist, individual results are presented for each of the 11 lists (PEP, DFAT, OFAC, etc) irrespective of state</p><p>One of two results are returned for each list:</p><p>NOT\_FOUND\_ON\_LIST</p><p>FOUND\_ON\_LIST</p> | <p>For the Dow Jones watchlist, if no matches are found, individual results with a state of **NOT\_FOUND\_ON\_LIST** will be returned for each watchlist category as below:</p><p>PEP</p><p>Extended PEPs</p><p>Sanctions</p> | <p>For the Dow Jones watchlist, if matches are found on the PEP watchlist or Extended PEP watchlist:</p><p>State of **MATCH\_REVIEW\_REQUIRED** will be returned</p><p>No further information is displayed about the match</p> | <p>For the Dow Jones watchlist, if matches are found on a Sanctions List:</p><p>State of **MATCH\_REVIEW\_REQUIRED** is returned</p><p>Name of the Sanctions list is returned - there can be one or more Sanctions List matches for an individual.</p> |
