# V 1.33 March 2019

## Release Dates

**Test**: Feb 19, 2019

**Production**: Mar 5, 2019

## In This Release

This release includes a new iOS SDK with improvements and bug fixes. The SDK is available from [http://greenidmobile.vixverify.com/](http://greenidmobile.vixverify.com/)

The iOS SDK release includes improvement to [customisation](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/336101657/iOS+UI+Customisation) on usability on the document review screens.

## iOS SDK{#ReleaseNotes:greenIDMobilev1.33-iOSSDK}

The greenID Mobile iOS SDK v1.33.0 is supported on iOS versions 10.0 – 11.3.1 (At the time of this release, 12.1 was the latest production release of iOS available.)

| **iOS SDK**       |                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Customisation** | <p>**Document review screen customisation**</p><p>Previously, the document review screens could not be configurable by clients. Now, we have made parts of the document review screens configurable on iOS. See [iOS UI Customisation section 4, Changing the document review screen UI](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/336101657/iOS+UI+Customisation#iOSUICustomisation-4.ChangingthedocumentreviewscreenUI)</p> |
| **Security**      | <p>**Moved information into a private header**</p><p>We have moved configuration and other non-essential information from a public header into a private header.</p>                                                                                                                                                                                                                                                                        |
| **Bug Fix**       | <p>**Viewing the document review screen using large font settings**</p><p>Previously, using an iPhone's large font settings would cause the text and '?' button on the document review screens to disappear. Now, we are able to display this text.</p>                                                                                                                                                                                     |

## Known Issues

No new constraints or known issues were introduced in this release.

Existing known issues can be found on the [Known issues in greenID Mobile](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/459835177/Known+issues+in+greenID+Mobile) page.
