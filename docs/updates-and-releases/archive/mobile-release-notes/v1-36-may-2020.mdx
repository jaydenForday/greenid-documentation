# V 1.36 May 2020

## Release Dates

**Test**: 18th May, 2020

**Production**: 18th May, 2020

## In This Release

This release includes a new Android SDK only. The new SDKs are available from [http://greenidmobile.vixverify.com/](http://greenidmobile.vixverify.com/)

## Android SDK{#ReleaseNotes:greenIDMobilev1.36-AndroidSDK}

The greenID Mobile Android SDK v2.9.2 is supported on Android: 8.0 – 10. (At the time of this release, 10.0 was the latest production release of Android available.)

| **Android SDK**        |                                                                                                                                                    |
| ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Accept new AID SDK** | <p>**AID capture framework update**</p><p>An update has been made to include updated versions of AID's document and selfie capture frameworks.</p> |

## Known Issues

No new Known Issues were introduced as part of this release. Existing known issues can be found on the [Known issues in greenID Mobile](https://vixverify.atlassian.net/wiki/spaces/GREEN/pages/459835177/Known+issues+in+greenID+Mobile) page.
